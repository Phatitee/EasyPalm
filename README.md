# EasyPalm
> #### คำสั่ง
>- run : npm start

>### Windows
>ดาวน์โหลด https://nodejs.org/
เลือก LTS (Long Term Support)
ดับเบิลคลิกติดตั้ง → ติ๊ก “Add to PATH”
>>>เช็คเวอร์ชัน:
node -v
npm -v 


>### Mac (2.ง่ายกว่า)
>1.ดาวน์โหลด https://nodejs.org/ 
เลือก LTS Mac Installer (.pkg)
ดับเบิลคลิกติดตั้ง
2.ติดตั้ง Node.js
brew install node@lts
>>>เช็คเวอร์ชัน:
node -v
npm -v
