# EasyPalm
#### คำสั่งใน Project
```
pip install flask flask-sqlalchemy flask-migrate flask-cors

run : npm start
python -m venv venv
pip install -r requirements.txt

.\venv\Scripts\activate

run backend (ใน ../backend):  
python seed.py
python run.py

run frontend (ใน ../frontend) :
npm install
npm start
```
